# ProductPulse PRD — Reduce Cart Abandonment & Improve Retention

## Goal
Increase **cart → purchase conversion** and improve **week-2 retention** using data-driven product improvements.

## Problem
Analytics show a major drop-off between **view → cart** and **cart → purchase**, indicating friction in product discovery and checkout.

## Users
- New shoppers exploring products
- Returning shoppers comparing options
- High-intent shoppers who add to cart but abandon

## Key Metrics (Success)
- View → Cart rate
- Cart → Purchase rate
- DAU/MAU
- Week-2 retention (cohort)
- Churn rate (14-day inactivity)

## Insights (from ProductPulse)
- Lowest conversion days: see `reports/funnel_daily.csv`
- Adoption vs retention: see `reports/adoption_retention.csv`
- Churn drivers: see `reports/churn_feature_importance.csv` 

## Proposed Solutions
### 1) Checkout Friction Reduction
- Reduce checkout steps (1-page checkout)
- Add trust elements (shipping/returns clarity)

### 2) Product Confidence Boost
- Reviews + ratings
- Similar items comparison

### 3) Activation Nudge
- Encourage early cart action with personalized prompts

## Experiments
- A/B test checkout redesign (see `reports/ab_test_summary.txt`)
- A/B test “reviews visible above fold” on PDP
